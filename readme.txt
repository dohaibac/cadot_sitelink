This Plugin help you to make referral link to other sites very easy with form and widget

== Description ==

* This plugin will add a SiteLink menu
* Create a form for add new 
* Have a table all current links with action delete
* This plugin include a Widget for display sitelink referral in sidebar.

Source code: https://github.com/dohaibac/cadot_sitelink

== Installation ==
This section describes how to install the plugin and get it working.
This plugin install very simple, as same as all other plugin. 

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress Admin


== Frequently Asked Questions ==

= How to add a new link referral? =

Please use Cadot siteLink menu, you will se the form at the top page. You enter the name of link and link address. The new link will insert to database and display in the widget.

= How about Horizontal line in widget? =

if you want to use horizontal line for separate link in widget, please select "Yes". If you selected "No", the widget will not have horizontal line.
